# babysharkui

## Install

```bash
npm install --save babysharkui
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'babysharkui'
import 'babysharkui/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
